
# JDBC Project (PreparedStatement)

## 📌 Project Overview

This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using Java JDBC with PreparedStatement.
It connects to a MySQL database and performs secure database operations.

## 🚀 Features

* Insert new user data into the database
* Fetch user details using ID
* Update existing user information
* Delete user records
* Uses PreparedStatement to prevent SQL Injection

## 🛠️ Technologies Used

* Java
* JDBC (Java Database Connectivity)
* MySQL
* Eclipse / IntelliJ IDE

## 📂 Project Structure

* `InsertUser.java` → Insert data into database
* `FetchUser.java` → Retrieve data from database
* `UpdateUser.java` → Update existing data
* `DeleteUser.java` → Delete records

## ⚙️ Database Configuration

Make sure you have:

* MySQL installed
* Database created (example: `user_details`)
* Table created (example: `user`)
